# 3.
MySQL은 오픈 소스 관계형 데이터베이스 관리 시스템(RDBMS)으로, 데이터를 저장하고 검색하는 데 사용됩니다.

MySQL 설치

1. sudo를 통해 설치해줍니다.
sudo apt install mysql-client-core-8.0

2. mysql -u admin -p -h 엔드포인트 로 접속합니다.
